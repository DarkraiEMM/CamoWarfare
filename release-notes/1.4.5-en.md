# CamoWarfare 1.4.5

This release focuses on Create Big Cannons balance and armor attachment behavior, and adds the first pass of vehicle roof hatches plus sloped attachment support.

## Changes

- Rebalanced camouflage hull and armor attachment durability for CBC combat.
- Added optional CBC compatibility in a dedicated mixin config so CamoWarfare does not require CBC to load.
- Added reactive armor behavior for add-on armor plates. CBC projectiles now break the reactive plate, play a small non-destructive blast effect, and are consumed before normal penetration or over-penetration can damage armor behind it.
- Changed slat armor behavior so cannon projectiles pass through cleanly, with a chance to reduce projectile mass and slightly reduce velocity.
- Reduced slat armor collision coverage so projectiles interact with sparse bars instead of the full visual panel.
- Restored Create Creative Worldshaper sampling for the decalable armor plate.
- Added vehicle deck hatches with trapdoor-style placement, metal sounds, redstone opening, waterlogging, color variants, recipes, translations, and tooltips.
- Added Copycats+ slope, slope-layer, and straight-stair placement support for add-on armor plates and slat armor.
- Added sloped slat armor models with lightweight collision and connector refresh logic for continuous grille runs.
- Added colored crafting recipes for add-on armor plates, slat armor, vehicle hanging plates, and vehicle deck hatches.
- Cleaned up shared Copycats surface detection and removed obsolete hatch texture leftovers.

## Notes

- CBC integration is optional and only activates when Create Big Cannons is installed.
- Reactive armor explosions are visual/feedback effects and should not damage surrounding blocks.
- Sloped attachment support targets Copycats+ slope-family blocks and keeps ordinary flat placement behavior intact.
- Minecraft 1.21.1 and NeoForge 21.1.227 are targeted.
