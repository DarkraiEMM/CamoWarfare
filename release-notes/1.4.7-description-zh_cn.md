# 迷彩战车介绍

迷彩战车是面向 Minecraft 1.21.1 NeoForge 的载具装饰与装甲细节模组，为战车、装甲车、舰船和军事风格建筑提供迷彩车体、外挂装甲、水贴和表面标识工具。

你可以使用标准/大块迷彩车体方块、附加装甲板、格栅装甲、车体挂载板、车厢舱盖、喷涂模板和旧化战术水贴来搭建更有层次的载具外观。创造栏按林地、山地、沙漠、雪地、夜战、海军、城市和纯色等主题整理，适合大型载具工程快速查找和搭配。

模组以多人载具建造整合包为目标设计。JEI、Copycats+ 和 Create Big Cannons 可作为可选兼容提供额外体验；即使未安装这些可选模组，迷彩战车本体也可以正常使用。

## 特色

- 多种地形主题的模块化迷彩车体方块
- 附加装甲板、格栅装甲、车体挂载板和车厢舱盖
- 用于编号、警示、识别和战术标记的水贴与喷涂模板
- 按主题整理的创造栏分区，方便快速搭建
- 可选 Copycats+ 与 Create Big Cannons 兼容
- 支持英文、简体中文、德语、法语和俄语本地化
