# 迷彩战车 1.4.5

本次更新主要调整 Create Big Cannons 环境下的装甲平衡和附件装甲行为，并加入第一版车厢舱盖与斜面附件支持。

## 更新内容

- 重新平衡迷彩车体方块和装甲附件在 CBC 战斗中的耐久表现。
- 将 CBC 兼容逻辑拆分到独立的可选 mixin 配置中，未安装 Create Big Cannons 时迷彩战车仍可正常加载。
- 为附加装甲板加入爆反行为。CBC 炮弹命中后会击碎爆反板，触发小型非破坏性爆炸效果，并在进入普通穿深或过穿结算前抵消炮弹。
- 调整格栅装甲行为。炮弹会顺畅穿过格栅，并有概率削减炮弹质量和少量速度。
- 缩小格栅装甲碰撞判定，让炮弹只与稀疏杆件交互，而不是碰到完整视觉面板就进入碰撞。
- 恢复 Create 创造环境改造枪对可贴花基础装甲板的取样识别。
- 新增车厢舱盖，支持原版活板门式放置、金属声音、红石开合、水含水状态，并补齐颜色变体、配方、翻译和提示文本。
- 为附加装甲板和格栅装甲新增 Copycats+ 斜坡、斜坡层、直楼梯贴附支持。
- 新增斜面格栅装甲模型、轻量碰撞和连接刷新逻辑，让格栅可以在连续斜面上衔接。
- 补齐附加装甲板、格栅装甲、车体挂载板和车厢舱盖的不同颜色合成配方。
- 清理 Copycats 斜面识别逻辑，改成共用附件支撑代码，并移除不再引用的舱盖残留贴图。

## 说明

- CBC 兼容为可选兼容，仅在安装 Create Big Cannons 时启用。
- 爆反触发的爆炸是反馈效果，不应破坏周围方块。
- 斜面附件支持面向 Copycats+ 斜坡系列方块，不改变普通平面放置行为。
- 目标版本为 Minecraft 1.21.1 与 NeoForge 21.1.227。
