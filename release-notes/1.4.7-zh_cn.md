# 迷彩战车 1.4.7

本次更新专注于联机和专用服务器加载稳定性。它将仅客户端使用的 GUI/渲染 mixin 从通用加载路径中拆出，并让可选 Copycats+ 兼容在未安装依赖时更安静、更安全。

## 更新内容

- 将创造栏 GUI mixin 拆分到独立的仅客户端 mixin 配置中。
- 将可选 Copycats+ 渲染 mixin 拆分到独立的仅客户端 mixin 配置中。
- 新增 Copycats+ mixin 插件，在应用兼容 mixin 前检查是否处于客户端以及目标类是否存在。
- 将创造栏分区标题渲染器移动到客户端包，避免 common/server 代码直接引用客户端 GUI 类。
- 将 Copycats+ 迷彩渲染状态从共享全局布尔值改为线程局部状态。
- 为 CamoWarfare 的 mixin 配置补充明确的 Mixin `minVersion` 元数据。

## 说明

- 推荐服务器和局域网联机玩家更新到此版本。
- 本版本不包含玩法平衡或内容改动。
- 此版本用于修复专用服务器日志中出现 `AbstractContainerScreen`、`CreativeModeInventoryScreen` 或 Copycats+ 模型类被服务端加载的问题。
- 目标版本为 Minecraft 1.21.1 与 NeoForge 21.1.227。
